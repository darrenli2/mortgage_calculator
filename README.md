## How to run the application

* `bundle install`
* `RAILS_ENV=development bundle exec rake db:reset`
* `bundle exec rails s`

## How to run specs
* `bundle install`
* `RAILS_ENV=test bundle exec rake db:reset`
* `bundle exec rspec`
